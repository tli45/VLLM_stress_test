## File Description

### performance_results.csv
- Detailed records of each stress test run
- Includes metrics such as concurrency level, latency, and success rate

### performance_thresholds.json
- Server performance threshold configuration
- Contains key data such as safe concurrency limits and crash points

## Workflow
1. Run the stress test → Results are automatically saved to `csv_results/`
2. Analyze results → Generate `threshold_analysis.csv`
3. Check logs → Troubleshoot and identify issues
